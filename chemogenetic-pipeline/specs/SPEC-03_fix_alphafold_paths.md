## Problem Statement

Three things are broken:

1. **AlphaFold outputs are in the wrong location.** Real CIF files from AlphaFold Server are in `data/structures/fold_hm3dq_*/fold_hm3dq_*_model_0.cif`, but `check_alphafold_outputs()` looks in `data/structures/predicted/` for files named `{name}.pdb` or `{name}.cif`.

2. **`predicted/` contains fake RDKit-generated PDBs.** The 6 files in `predicted/` are ~4KB small-molecule conformers from `generate_demo_structures()`, NOT protein structures. They need to be replaced.

3. **pLDDT scores are synthetic random numbers.** Files in `data/results/plddt_scores/` were generated by `generate_demo_plddt_scores()` using `np.random.normal()`. All residues are labeled "ALA".

if results[compound] is None:
    safe_lower = safe_name.lower()
    fold_pattern = os.path.join(structures_dir, f"fold_hm3dq_{safe_lower}")
    if os.path.isdir(fold_pattern):
