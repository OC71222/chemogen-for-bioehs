### Step 2: Verify DCZ analysis is real data

Check if `data/results/md_analysis_DCZ.csv` exists and whether it's real or synthetic:

```python
import pandas as pd
import os

csv_path = "data/results/md_analysis_DCZ.csv"
if os.path.exists(csv_path):
    df = pd.read_csv(csv_path)
    print(f"Rows: {len(df)}")
    print(f"Columns: {list(df.columns)}")
    print(f"RMSD range: {df['protein_rmsd_A'].min():.2f} - {df['protein_rmsd_A'].max():.2f} Å")

    # Check for synthetic signature: monotonically increasing random walk
    diffs = df['protein_rmsd_A'].diff().dropna()
    pct_increasing = (diffs > 0).mean()
    print(f"% frames where RMSD increases: {pct_increasing:.2f}")
    # Real MD: should fluctuate around equilibrium (~50% increasing)
    # Synthetic random walk: >60% increasing over long timescales
else:
    print("DCZ analysis CSV does not exist — needs re-analysis")
```

### Step 4: Run MD for remaining 5 compounds

Each MD simulation is computationally expensive. Run one at a time:

```bash
# Each takes significant time depending on simulation length
python3 -m src.module8.run_module8 --compound CNO --length-ns 50
python3 -m src.module8.run_module8 --compound Clozapine --length-ns 50
python3 -m src.module8.run_module8 --compound "Compound 21" --length-ns 50
python3 -m src.module8.run_module8 --compound Olanzapine --length-ns 50
python3 -m src.module8.run_module8 --compound Perlapine --length-ns 50
```

**Expected per simulation:**
- System setup: 2-10 minutes (solvation, parameterization)
- Minimization + equilibration: 5-30 minutes
- 50ns production MD: hours to days depending on GPU/CPU
  - With GPU: ~2-8 hours per compound
  - CPU only: ~24-72 hours per compound

**Consider shorter simulations for initial testing:**
```bash
python3 -m src.module8.run_module8 --compound CNO --length-ns 5
```
